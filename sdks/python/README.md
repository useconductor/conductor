# sdks/python

Python SDK for Conductor. Provides a Python client for interacting with the Conductor MCP server.

## Contents

- `pyproject.toml` - Python package manifest and dependencies
- `conductor/` - Python package source code

## Architecture

This SDK enables Python applications to communicate with Conductor via the MCP protocol. Install as a pip package to use Conductor tools from Python scripts, Jupyter notebooks, or AI agent frameworks.
